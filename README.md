# Mpeg4AddChapterTool

## 概要
Mpeg4の動画ファイルにチャプター情報を追加するツールです。ソフトウェアの使用には別途[MP4Box](https://gpac.wp.imt.fr/)のexeが必要です。

## 留意事項
各ファイルへのパスが英数字以外だったり、パスが長すぎたりすると、MP4BoxでI/Oエラーが発生し、正常に処理できない場合があります。
